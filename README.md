# Portfolio Project: Mixed Messages

### Description
This is the first portfolio project for the CodeCademy back-end developer career course. This project is a message generating system that uses JavaScript to ask a user what they want to do.
Based on that input, the program can do a number of things.

+ Allows user to view all available classes, weapons, and items
+ Allows a user to add a class, weapon, or item
+ Randomly selects a loadout for the user with a class, weapon, and item
